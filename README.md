# AI-Powered Chatbot Assistant

A collection of Python-based chatbot implementations with various functionalities, including voice interaction, web-based interfaces, and OpenAI-powered conversation.

## Features

1. **Voice Assistant (AI Assistant.py)**
   - Voice-enabled interaction using pyttsx3
   - Multiple system operations (Notepad, Spotify, etc.)
   - Time-based greetings
   - Web search capabilities
   - Email functionality

2. **Web Interface Chatbot**
   - Browser-based chat interface
   - Easy deployment
   - Interactive web experience

3. **OpenAI-Powered Chatbot**
   - Advanced conversation capabilities
   - Customizable chatbot personality
   - Integration with GPT-3.5-turbo
   - Continuous conversation history

## Requirements

- Python 3.x
- Required Python packages:
  - pyttsx3
  - datetime
  - pywhatkit
  - AppOpener
  - requests
  - subprocess
  - smtplib
  - wikipedia
  - googlesearch
  - openai

## Installation

1. Clone the repository
2. Install the required packages:
   ```bash
   pip install pyttsx3 pywhatkit AppOpener wikipedia googlesearch openai
   ```

## Usage

1. **Voice Assistant**
   ```bash
   python AI Assistant.py
   ```

2. **OpenAI Chatbot**
   ```bash
   python "Interactive chatbot using OpenAI API.py"
   ```
   - Replace "YOUR_API_KEY" with your OpenAI API key
